My First Webpage

This is my very first web development project! It serves as a personal introduction page built entirely with HTML5.

Project Description

This project was created to learn the fundamentals of web structure and HTML tags. It displays my personal profile, contact information, and my motivations for learning web development.

Features

Semantic HTML Structure: Uses header, section, and footer tags for better accessibility.

Personal Bio: A brief introduction about myself (Sreenivasan S).

Learning Goals: Three detailed paragraphs explaining my journey into coding.

Clean Layout: Structured text without external styling dependencies.

How to View

Download or clone this repository.

Locate the index.html file in the folder.

Double-click index.html to open it in your default web browser (Chrome, Firefox, Safari, etc.).
